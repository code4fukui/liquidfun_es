# liquidfun_es

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

LiquidFun is a particle-based fluid simulation library, wrapped in JavaScript and visualized using D3.js.

## Demo

- [IchigoJam logo](https://code4fukui.github.io/liquidfun_es/)
- [sample](https://code4fukui.github.io/liquidfun_es/sample.html) (forked https://qiita.com/Quramy/items/578efec667267acf6871)

## Features

- Particle-based fluid simulation
- Flexible object shapes
- Damping and other physics parameters

## Requirements

None, runs in modern web browsers.

## Usage

Open the HTML files in a web browser.
